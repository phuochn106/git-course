Câu 1:
    git commit --amend
    => Dùng để chỉnh sửa lại commit message mới nhất

    git reset HEAD~3
    => undo lại 3 commit cuối cùng trên vùng repository. Code của 3 lần commit này vẫn còn trên vùng working
    (remove 3 commit cuối khỏi vùng repository)
